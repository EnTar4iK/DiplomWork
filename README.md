# DiplomWork

Интернет-магазин электроники на PHP и MySQL/MariaDB.

## Запуск проекта

1. Установите PHP 8.x и MySQL или MariaDB.
2. Создайте базу данных `PP2`.
3. Импортируйте дамп `pp2/PP2.sql` в базу `PP2`.
   - Если используете MariaDB и импорт ругается на collation `utf8mb4_0900_ai_ci`, замените её на `utf8mb4_unicode_ci`.
4. Проверьте подключение в `pp2/config/db.php`:

```php
$host = "localhost";
$login = "root";
$password = "";
$db_name = "PP2";
```

5. Запустите локальный сервер из корня проекта:

```bash
php -S 127.0.0.1:8000 -t pp2
```

6. Откройте сайт:

```text
http://127.0.0.1:8000/index.php
```

## Тестовые данные

Дамп `pp2/PP2.sql` содержит товары, пользователей и заказы. Для входа в админ-панель можно использовать:

```text
Логин: admin
Пароль: admin
```
