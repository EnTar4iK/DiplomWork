# DАЙКОМ Store

Учебный проект на **голом PHP + MySQL**: современный сайт магазина электроники с каталогом, карточками товаров, корзиной, оформлением заказа, оплатой, доставкой, личным кабинетом и админ-панелью.

## Что внутри

- `pp2/` — полноценная PHP-версия сайта.
- `pp2/PP2.sql` — готовая база данных для импорта через phpMyAdmin.
- `pp2/pages/` — статическое демо для GitHub Pages. GitHub Pages не выполняет PHP/MySQL, поэтому серверная логика работает только в PHP-окружении.

## Запуск PHP-версии

1. Создайте базу данных `PP2` в phpMyAdmin.
2. Импортируйте файл `pp2/PP2.sql`.
3. Проверьте доступы в `pp2/config/db.php`:
   - host: `localhost`
   - user: `root`
   - password: пустой
   - database: `PP2`
4. Откройте `pp2/index.php` через локальный PHP-сервер, OpenServer, XAMPP или MAMP.

Демо-аккаунты:

- Админ: `admin` / `admin`
- Покупатель: `client` / `12345`

## GitHub Pages

Для GitHub Pages используется статическая витрина из `pp2/pages/`. Она показывает дизайн, товары и интерактивную демо-корзину без PHP/MySQL.
